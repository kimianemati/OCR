Persian License Plate OCR:
A Python-based Optical Character Recognition (OCR) system for recognizing Iranian (Persian) vehicle license plates using image preprocessing and EasyOCR.

Features:
-Persian (Farsi) license plate recognition

-Image preprocessing pipeline for improved OCR accuracy
-Contrast enhancement using CLAHE

-Adaptive thresholding

-Multiple OCR passes on different preprocessed images

-Confidence-based result selection

-Persian character filtering
